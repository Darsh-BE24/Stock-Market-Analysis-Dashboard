# Stock Market Analysis Dashboard

An interactive financial analytics dashboard built using Python and Power BI to analyze stock market trends, technical indicators, volatility, and risk-adjusted performance.

The project uses real-time market data fetched through Yahoo Finance and transforms it into meaningful financial insights through data preprocessing, technical indicators, and interactive visualizations.

---

# Features

- Comparative stock performance analysis
- RSI (Relative Strength Index) analysis
- Moving Average Analysis (MA7, MA30, EMA20)
- Volatility tracking
- Sharpe Ratio analysis
- Technical trend analysis
- Interactive stock and date filters
- Market-wide and stock-specific analytics

---

# Technical Indicators Used

| Indicator | Description |
|---|---|
| Daily Return % | Measures daily stock movement |
| Volatility % | Measures price fluctuation |
| MA7 | 7-Day Moving Average |
| MA30 | 30-Day Moving Average |
| EMA20 | 20-Day Exponential Moving Average |
| RSI | Momentum indicator for overbought/oversold conditions |
| Rolling Volatility | Dynamic market risk indicator |
| Sharpe Ratio | Risk-adjusted return metric |
